Tools for astronomical data analysis mostly for 
Gamma-Ray Bursts observed with GROND 

bb_sed.py: Broadband SED Plotting; similar to SED:bbplot.py
get_ebv.py: get E_B-V for given coordinates
get_nh.py: get NH for given coordinates
get_corr.py: get_ebv.py & get_nh.py combined
fc_plot.py: Create Paper-like Findingcharts
